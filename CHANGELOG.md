# Changelog

## 0.1.0 (2026-05-09)

### Initial Release
- Project scaffolded with Flutter.
- Core data models (Note, Tag) with Hive adapters.
- Centralized app theme with Merriweather and Inter fonts.
- Navigation structure using `go_router` and Bottom Navigation Bar.
- Note list and CRUD logic with Riverpod.
- Rich text editor integration with `flutter_quill` and auto-save.
- Search functionality and tags management.
