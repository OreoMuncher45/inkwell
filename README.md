# Inkwell Notes

A premium, editorial-style, offline-first note-taking app built with Flutter.

## Features
- **Immersive Writing:** Rich text editor powered by `flutter_quill`.
- **Offline-First:** Local storage using `Hive`.
- **Structured Organization:** Tags and search functionality.
- **Modern UI:** Material 3 design with brown color scheme and premium typography (Merriweather & Inter).
- **State Management:** Powered by `Riverpod`.
- **Responsive:** Works on Web and Android.

## Development
```bash
flutter run -d chrome
```
